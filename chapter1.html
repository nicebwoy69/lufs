Chapter 1: Functions, Rules, and Common Mistakes

1.1 Introduction to Functions

In calculus, a function describes how one quantity depends on another. A function, written as $f(x)$, shows how the dependent variable $f(x)$ changes as you adjust the independent variable $x$. Functions are fundamental in audio processing for representing time-dependent audio signals.

1.2 Types of Functions and Their Rules

Linear Functions

Linear functions take the form:

$y = mx + b$

  • $m$ is the slope, indicating how steep the line is.
  • $b$ is the y-intercept, showing where the line crosses the y-axis.

Common Mistakes:

  • Confusing the slope and y-intercept values.
  • Assuming the slope is zero when none is specified, whereas the default is $m = 1$.

Example:

To graph $y = 3x - 2$:

  1. Start by plotting the y-intercept at $(0, -2)$.
  2. Since the slope is 3, move up 3 units and right 1 unit from $(0, -2)$ to $(1, 1)$.
  3. Draw a straight line through these points.

Quadratic Functions

The general form of a quadratic function is:

$y = ax^2 + bx + c$

  • If $a > 0$, the parabola opens upward.
  • If $a < 0$, the parabola opens downward.

Common Mistakes:

  • Errors in applying the quadratic formula or factoring incorrectly.

Example:

For $y = x^2 - 4x + 4$, factor it as $y = (x - 2)^2$ to see the vertex is at $(2, 0)$. Plot this and additional points to sketch the parabola.

Exponential Functions

The general form of an exponential function is:

$y = ae^{bx}$

  • $e$ is Eulerâ€™s number, approximately 2.718.
  • If $b > 0$, the function grows exponentially; if $b < 0$, it decays exponentially.

Common Mistakes:

  • Confusing $e^{bx}$ with $bx \cdot e$.

Logarithmic Functions

Logarithmic functions have the form:

$y = \log_b(x)$

  • $b$ is the logarithm base. Common bases include 2, 10 (for decibels), and $e$.

Example:

Calculating the decibel level of a sound 10 times as intense as a reference level:

$dB = 10 \log_{10}(10) = 10 \, dB$

Trigonometric Functions

Trigonometric functions model periodic behavior, vital in audio processing for waveforms:

  • $y = \sin(x)$
  • $y = \cos(x)$
  • $y = \tan(x)$

Common Mistakes:

  • Using degrees instead of radians in calculus.
  • Misinterpreting phase shifts in sine and cosine waves.

Example:

To plot $y = \sin(x)$ from $0$ to $2\pi$:

  1. At $x = 0$, $y = 0$. Plot $(0, 0)$.
  2. At $x = \pi$, $y = 0$. Plot $(\pi, 0)$.
  3. Continue for $2\pi$, observing the periodic pattern.

Piecewise Functions

Piecewise functions use different rules over different intervals, helpful in audio processing for segmented changes over time.

Example:


f(x) = {
    x + 2,  if x < 0
    -x + 3, if x >= 0
}

1.3 Working with Derivatives

Derivatives describe how quickly a function changes, fundamental for tracking changes in audio over time.

Power Rule:

$\frac{d}{dx} x^n = nx^{n-1}$

For example, the derivative of $x^3$ is $3x^2$.

Product Rule:

$\frac{d}{dx} [u(x) v(x)] = u'(x)v(x) + u(x)v'(x)$

Chain Rule:

$\frac{d}{dx} f(g(x)) = f'(g(x)) \cdot g'(x)$

This rule helps differentiate nested functions.

Example of Applying the Chain Rule:

If $f(x) = (3x^2 + 2)^4$, let $g(x) = 3x^2 + 2$. Find $f'(x)$ by first differentiating the outer and inner functions:

$f'(x) = 24x(3x^2 + 2)^3$

1.4 Practice Problems

  • Differentiate $f(x) = 3x^4 + 2x^2$.
  • Use the product rule for $f(x) = (x^2)(x + 3)$.
  • Differentiate $f(x) = \sin(3x)$.
  • Solve $f(x) = (2x^3 + 5x)^2$ using the chain rule.
